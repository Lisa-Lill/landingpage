A tool for the discovery of candidate genes controlling complex traits

QTLNetMiner is a user-friendly web application that can interrogate plant and animal knowledge networks and be used to show candidate genes and QTL associated with given input terms (e.g. early flowering, disease resistance). The relevance of a gene to particular query terms is weighted using information retrieval and network inference methods. The supporting evidence networks for selected candidate genes can be visualized in the new, lightweight KNETviewer (or in the Ondex Web Java-applet). QTLNetMiner is designed in a generic way and can be created for any organism with an integrated Ondex knowledge network (www.ondex.org). New QTLNetMiner video now on
